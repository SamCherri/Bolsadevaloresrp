# Checklist de UI/Responsividade

## Breakpoints obrigatórios
- [ ] mobile pequeno (~320px)
- [ ] mobile normal (~390px)
- [ ] tablet (~768px)
- [ ] desktop (~1024px)
- [ ] desktop largo (>=1440px)

## Itens de revisão
- [ ] Sidebar navegável em mobile.
- [ ] Topbar sem corte de texto e com botão tocável.
- [ ] Tabelas com `overflow-x-auto` e `min-width`.
- [ ] Cards sem overflow horizontal.
- [ ] Formulários com botões full-width no mobile.
- [ ] Grids empilham corretamente em telas pequenas.
- [ ] Histórico (ordens/câmbio/trades) quebra linha sem truncar dados.
- [ ] Empty states visíveis quando listas estiverem vazias.
